>标记的含义

白色字体是基础、必须会
白字带🌟表示重中之重，最好能讲清楚原理或自己实现

红色字体代表有多余的时间再看
红字带🌟可以不看

PS：红线代表不影响学习进度，可以先跳过，后面在学
